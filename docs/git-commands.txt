Git commands

1. CLONE REPOSITORY - get a new copy of the repository on your machine

    git clone git@github.com:mihaisarbu/cavu-training.git

2. CREATE NEW BRANCH - create a new local branch

    git checkout -b <author name>/<issue name>

3. CHECK STATUS - check the status of all new, modified or removed files

    git status

4. ADD FILES - add new or edited files to be commited

    - add specific file:
    git add <file>  

    - add all new and modified files:
    git add .

5. COMMIT CHANGES - commit all the added files

    git commit -m "<commit message>"

6. ADD AND COMMIT MODIFIED FILES - shortcut to add all the modified files and commit them

    git commit -a -m "<commit message>"

7. PUSH CHANGES - push all the commited files

    - initial push:
    git push --set-upstream origin <author name>/<issue name>

    - subsequent pushes:
    git push
